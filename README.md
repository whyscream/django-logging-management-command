# Django logging management command

A proof of concept / reference implementation of a Django management command base class that supports logging and verbosity handling by default.

For details, see https://forum.djangoproject.com/t/implement-default-handling-of-verbosity-in-management-commands/44892

# Setup

`uv sync` should setup a working environment for developing or testing.

# Testing

Use `pytest` to run the included tests.
